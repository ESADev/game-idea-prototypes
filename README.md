# game-idea-prototypes

## Grid Clash prototype

Open `/index.html` in a browser (or deploy to Vercel as a static site).

### Gameplay
- Large grid area-control match (starts at 15x15)
- Player and enemy place marks to control more cells before the board fills
- Form 3-in-a-row to gain special attack charges
- Spend a turn to use a charge:
  - **Radial clear**: remove enemy marks in a circular area
  - **Laser line** (unlockable): clear one full row or column
- Between matches:
  - Gain strategic points on win
  - Buy upgrades (laser, double mark turn)
  - Each win increases difficulty and grid size
