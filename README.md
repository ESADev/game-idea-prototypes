# game-idea-prototypes

## Chess Survivor Prototype

Fast action survival on a chess-like grid.

- Pick one player piece (Knight, Rook, or Bishop).
- Move/capture with chess movement rules.
- Avoid dark red danger squares (enemy attack map).
- Survive enemy waves, capture enemies, and collect power-ups.

### Run

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```

### Controls

- Mouse: click highlighted destination squares.
- Keyboard: WASD or Arrow keys to move cursor, Enter/Space to move.
