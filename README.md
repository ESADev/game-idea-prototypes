# game-idea-prototypes

## Tactical Bowling Prototype (Vercel-friendly web app)

A simple turn-based tactical bowling proof-of-concept.

### Run locally

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
npm run preview
```

## Gameplay loop

- Start a run.
- Choose start column, throw angle, and ball type.
- Throw to damage and push enemy pins.
- Surviving enemies advance one row each turn.
- Clear all incoming waves before enemies reach your line.
- Spend earned currency between runs on permanent upgrades.
