# game-idea-prototypes

## Pac-Man but DOOM prototype

Static web prototype implemented with HTML canvas.

### Run locally

Open `index.html` in a browser.

### Controls

- Move: `WASD`
- Aim + Shoot: `Arrow Keys` (hold) or `Mouse` (hold left click)
- Shop: `1/2/3` buy upgrades, `Enter`/`Space` start next run

### Loop

Collect points in the maze, survive enemy swarms, die, buy permanent upgrades, and restart.
